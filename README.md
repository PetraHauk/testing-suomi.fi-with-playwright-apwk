## Installation

### Prerequisites
- Ensure you have Java and Maven installed.

### Setup Instructions
1. Clone the repository.
2. Install dependencies:

   ```bash
   mvn dependency:resolve
   npx playwright install
   ```
3. Run tests:

   ```bash
   mvn test
   ```
