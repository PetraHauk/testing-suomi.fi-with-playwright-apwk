### Setup Instructions
1. Clone the repository.
```
git clone http://
``'
2. Tarkista ja asennetaan dependencies:

   ```bash
   mvn dependency:resolve
   npx playwright install
   ```
3. Asenna tarvittavat npm-paketit:

   ```bash
   npm install
   ```
4. To run the tests,:
   ```bash
   npm test
   ```
5. start appllication:

   ```bash
   npm start
   ```
